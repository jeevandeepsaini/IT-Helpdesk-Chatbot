# VPN Setup Guide

## Overview
This guide will help you set up and connect to the corporate VPN from any location.

## Prerequisites
- Active employee account
- VPN credentials (username and password)
- Stable internet connection

## Installation Steps

### Windows
1. Download the VPN client from the IT portal at https://it.company.com/vpn
2. Run the installer as Administrator
3. Follow the installation wizard
4. Restart your computer when prompted

### macOS
1. Download the VPN client for macOS from the IT portal
2. Open the .dmg file
3. Drag the VPN app to Applications folder
4. Grant necessary permissions in System Preferences > Security & Privacy

## Configuration

1. Launch the VPN client application
2. Click "Add New Connection"
3. Enter the following details:
   - Server: vpn.company.com
   - Port: 443
   - Protocol: OpenVPN
4. Enter your employee credentials
5. Save the configuration

## Connecting to VPN

1. Open the VPN client
2. Select the corporate network profile
3. Click "Connect"
4. Wait for the connection to establish (usually 10-15 seconds)
5. You should see a green "Connected" status

## Troubleshooting

### Cannot Connect
- Verify your internet connection is working
- Check that your credentials are correct
- Ensure the VPN client is up to date
- Try restarting the VPN client

### Slow Connection
- Disconnect and reconnect
- Try a different VPN server if available
- Check your local internet speed

### Connection Drops Frequently
- Update the VPN client to the latest version
- Check firewall settings
- Contact IT support if issue persists
