# Password Reset Guide

## Self-Service Password Reset

### For Active Directory Accounts

1. Go to https://password.company.com
2. Click "Forgot Password"
3. Enter your employee ID or email address
4. Choose verification method:
   - SMS to registered mobile
   - Email to alternate address
   - Security questions
5. Enter the verification code received
6. Create a new password following these requirements:
   - Minimum 12 characters
   - At least one uppercase letter
   - At least one lowercase letter
   - At least one number
   - At least one special character (!@#$%^&*)
   - Cannot reuse last 5 passwords
7. Confirm the new password
8. Click "Reset Password"

### Password Requirements
- Length: 12-64 characters
- Complexity: Must include uppercase, lowercase, numbers, and special characters
- Expiration: Passwords expire every 90 days
- History: Cannot reuse last 5 passwords

## If Self-Service Fails

Contact IT Support:
- Email: support@company.com with subject "Password Reset Request"
- Phone: ext. 5555
- In-person: Visit IT desk on Floor 3

Provide:
- Employee ID
- Full name
- Department
- Manager name

## Account Lockout

If your account is locked after multiple failed login attempts:
1. Wait 30 minutes for automatic unlock
2. OR contact IT Support for immediate unlock
3. Verify you're using the correct password

## Multi-Factor Authentication (MFA) Reset

If you need to reset your MFA device:
1. Go to https://mfa.company.com
2. Click "Reset MFA Device"
3. Verify identity using backup codes
4. Follow prompts to register new device

## Tips for Strong Passwords
- Use a passphrase (e.g., "Coffee!Morning@2024")
- Use a password manager
- Never share your password
- Don't write passwords down
- Use different passwords for different systems
