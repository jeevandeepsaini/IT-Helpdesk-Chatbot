# Email Configuration Guide

## Outlook Desktop Setup

### Windows Outlook

1. Open Microsoft Outlook
2. Click File > Add Account
3. Enter your email address: firstname.lastname@company.com
4. Click "Connect"
5. Enter your password when prompted
6. Complete MFA verification
7. Wait for account setup to complete
8. Click "Done"

### macOS Mail

1. Open Mail application
2. Go to Mail > Add Account
3. Select "Microsoft Exchange"
4. Enter your details:
   - Name: Your full name
   - Email: firstname.lastname@company.com
   - Password: Your AD password
5. Click "Sign In"
6. Complete MFA if prompted
7. Select what to sync (Mail, Contacts, Calendars)

## Mobile Email Setup

### iOS (iPhone/iPad)

1. Go to Settings > Mail > Accounts
2. Tap "Add Account"
3. Select "Microsoft Exchange"
4. Enter your email and description
5. Tap "Next"
6. Enter password and complete MFA
7. Choose what to sync
8. Tap "Save"

### Android

1. Open Gmail or Email app
2. Tap Menu > Settings > Add Account
3. Select "Exchange and Office 365"
4. Enter email address
5. Tap "Manual Setup"
6. Enter password
7. Server: outlook.office365.com
8. Complete setup

## Email Signature Setup

1. In Outlook, go to File > Options > Mail
2. Click "Signatures"
3. Click "New"
4. Enter signature name
5. Type your signature:
   ```
   [Your Name]
   [Job Title]
   [Department]
   Company Name
   Phone: [Your Extension]
   Email: [Your Email]
   ```
6. Click "OK"

## Common Issues

### Cannot Send/Receive Emails
- Check internet connection
- Verify credentials are correct
- Ensure Outlook is up to date
- Check mailbox quota

### Emails Going to Spam
- Add sender to Safe Senders list
- Check spam filter settings
- Contact IT if legitimate emails are blocked

### Sync Issues
- Remove and re-add account
- Clear Outlook cache
- Check server settings

## Email Best Practices
- Check email at least twice daily
- Use clear subject lines
- Reply within 24 hours
- Use professional language
- Don't send sensitive data via email
