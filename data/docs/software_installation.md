# Software Installation Guide

## Standard Software Catalog

Access the Software Center to install approved applications:
- Windows: Start Menu > Software Center
- macOS: Applications > Self Service

## Common Software Installations

### Microsoft Office Suite

**Windows:**
1. Open Software Center
2. Search for "Microsoft Office"
3. Click "Install"
4. Wait for installation (15-20 minutes)
5. Launch any Office app
6. Sign in with company credentials

**macOS:**
1. Open Self Service
2. Find "Microsoft Office for Mac"
3. Click "Install"
4. Enter admin password if prompted
5. Sign in with company credentials

### Web Browsers

**Chrome:**
1. Download from Software Center
2. Install automatically
3. Set as default browser (optional)

**Firefox:**
1. Available in Software Center
2. Click Install
3. Launch and configure

### Adobe Acrobat Reader

1. Search in Software Center
2. Click "Adobe Acrobat Reader DC"
3. Install
4. Set as default PDF viewer

### Zoom

1. Find Zoom in Software Center
2. Click Install
3. Launch Zoom
4. Sign in with company SSO

## Custom Software Requests

For software not in the catalog:

1. Submit request at https://it.company.com/software-request
2. Provide:
   - Software name and version
   - Business justification
   - License information (if known)
   - Manager approval
3. Wait for IT approval (2-3 business days)
4. IT will install or provide instructions

## Software Updates

### Windows
- Updates deploy automatically via WSUS
- Restart when prompted
- Critical updates install overnight

### macOS
- Check Software Update regularly
- Install company-approved updates
- Restart as needed

## Troubleshooting

### Installation Fails
- Ensure you have admin rights
- Check disk space (need 10GB free minimum)
- Restart computer and retry
- Contact IT if persists

### Software Not in Catalog
- Submit software request
- Provide business justification
- Wait for approval

### License Issues
- Verify you're signed in with company account
- Check license allocation with IT
- Request license if needed

## Prohibited Software
Do not install:
- Unauthorized file sharing tools
- Unapproved cloud storage
- Pirated software
- Personal VPN clients
- Cryptocurrency miners

Violations may result in disciplinary action.
