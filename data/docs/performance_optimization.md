# System Performance Optimization

## Windows Performance Optimization

### Check System Performance

1. Open Task Manager (Ctrl+Shift+Esc)
2. Click "Performance" tab
3. Check:
   - CPU usage
   - Memory usage
   - Disk usage
   - Network usage

### Free Up Disk Space

**Disk Cleanup:**
1. Search "Disk Cleanup"
2. Select drive (usually C:)
3. Check items to delete:
   - Temporary files
   - Downloads folder
   - Recycle Bin
   - Thumbnails
4. Click "Clean up system files"
5. Check "Previous Windows installations"
6. Click OK

**Storage Sense:**
1. Settings > System > Storage
2. Turn on "Storage Sense"
3. Configure automatic cleanup
4. Run now if needed

### Improve Startup Time

**Disable Startup Programs:**
1. Task Manager > Startup tab
2. Right-click unnecessary programs
3. Click "Disable"
4. Restart computer

**Common programs to disable:**
- Spotify
- Skype (if not used)
- Adobe updaters
- Manufacturer bloatware

### Optimize Memory

**Close Unnecessary Programs:**
1. Check Task Manager > Processes
2. Sort by Memory
3. Close high-memory programs not in use

**Increase Virtual Memory:**
1. System Properties > Advanced
2. Performance > Settings
3. Advanced > Virtual Memory
4. Change
5. Set custom size (1.5x RAM)

### Speed Up Computer

**Disable Visual Effects:**
1. System Properties > Advanced
2. Performance > Settings
3. Select "Adjust for best performance"
4. Or customize:
   - Keep "Show thumbnails"
   - Keep "Smooth edges of screen fonts"
   - Disable others

**Defragment Hard Drive:**
(Only for HDD, not SSD)
1. Search "Defragment"
2. Select drive
3. Click "Optimize"
4. Schedule weekly optimization

### Update Drivers

1. Device Manager
2. Right-click device
3. Update driver
4. Search automatically

**Priority drivers:**
- Graphics card
- Network adapter
- Chipset

### Scan for Malware

1. Windows Security
2. Virus & threat protection
3. Quick scan
4. Or Full scan if issues persist

## macOS Performance Optimization

### Check Activity Monitor

1. Applications > Utilities > Activity Monitor
2. Check CPU, Memory, Disk, Network tabs
3. Identify resource-heavy processes

### Free Up Storage

**Storage Management:**
1. Apple menu > About This Mac > Storage
2. Click "Manage"
3. Options:
   - Store in iCloud
   - Optimize Storage
   - Empty Trash Automatically
   - Reduce Clutter

**Manual Cleanup:**
- Delete old downloads
- Remove unused applications
- Clear cache: ~/Library/Caches
- Delete old iOS backups

### Reduce Startup Items

1. System Preferences > Users & Groups
2. Login Items tab
3. Select items to remove
4. Click "-"

### Reset SMC and NVRAM

**Reset SMC:**
(Fixes power, battery, thermal issues)
1. Shut down Mac
2. Press Shift+Control+Option+Power for 10 seconds
3. Release and start normally

**Reset NVRAM:**
1. Shut down Mac
2. Start and immediately press Option+Command+P+R
3. Hold for 20 seconds
4. Release and let Mac start

### Optimize Memory

**Reduce Memory Pressure:**
- Close unused applications
- Limit browser tabs
- Quit memory-heavy apps when not needed
- Restart Mac weekly

## General Tips

### Browser Optimization

**Chrome/Edge:**
- Clear cache and cookies
- Disable unused extensions
- Use fewer tabs
- Enable hardware acceleration

**Settings:**
1. Settings > Privacy and security
2. Clear browsing data
3. Select time range
4. Clear data

### Email Optimization

**Outlook:**
- Archive old emails
- Compact mailbox
- Disable auto-download of images
- Limit sync range to 3 months

### Network Optimization

- Use wired connection when possible
- Close bandwidth-heavy applications
- Disable cloud sync during important tasks
- Clear DNS cache

## When to Request Hardware Upgrade

Contact IT if:
- Consistent high CPU usage (>80%)
- Memory usage always >90%
- Disk constantly at 100%
- Computer >5 years old
- Frequent crashes or freezes

Submit request: https://it.company.com/hardware-request

## Preventive Maintenance

**Weekly:**
- Restart computer
- Clear browser cache
- Check for updates

**Monthly:**
- Run disk cleanup
- Review startup programs
- Scan for malware
- Check disk space

**Quarterly:**
- Review installed programs
- Update all drivers
- Clean physical dust (desktop)
- Backup important files

## Support
